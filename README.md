# Calendar App/Timer App

This is a calendar and timer widget app, where there is an up-to-date calendar as well as a functioning timer. It is a simple project as it is my first, but I have now polished and updated it to be more appealing and aesthetic. Thank you for reading!!!

This is my first-ever app, and I had so much fun making it. I followed a tutorial from @Nashallery from their YouTube channel. Hope you enjoy the app :) Had trouble with debugging, so I used 2 minutes of Copilot to try to understand what was wrong. Unfortunately, it didn't help me with debugging, and instead, I used debugging in the VS Code terminal.

I used the tutorial just to start my Electron React app and get the basics of a calendar function. After I wanted to add more and use my own new knowledge, I added a timer and some sound effects. It isn't similar to the tutorial that much, and I used my own styling and added a timer feature myself using some research. Here is the tutorial for reference and credit: https://www.youtube.com/watch?v=btxGSJ3Dh8E
